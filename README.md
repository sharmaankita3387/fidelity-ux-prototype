# Fidelity Canada | Digital App Dev Prototype

### Project Overview
This repository contains a **Smart Investor Profiler** built as a proof-of-concept for the 2026 Digital Applications Development student role. The tool analyzes user behavior (investment goals) and utilizes a classification model to recommend a portfolio strategy.

### Tech Stack & Skills Demonstrated
* **Frontend:** HTML5, CSS3 (Custom Variables, Grid/Flexbox), jQuery.
* **Data Viz:** Chart.js for real-time asset allocation rendering.
* **Engineering:** State persistence (localStorage), Asynchronous logic (Simulated API Fetch), Responsive Design.
* **UX/UI:** Dark Mode support, simulated AI processing states.

### Alignment with Fidelity Job Duties
1.  **Analyze Behavior:** Implemented a classification model to group users into risk profiles.
2.  **Responsive Design:** Mobile-first architecture using CSS Grid.
3.  **Next-Gen Investigation:** Explored state persistence and dynamic data visualization.

### How to Run
1. Clone the repository.
2. Open `index.html` in a browser (use Live Server for AJAX features).
